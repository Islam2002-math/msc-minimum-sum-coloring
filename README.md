# Minimum Sum Coloring (MSC) with Tabu Search

Projet Python pour expérimenter la **Coloration Somme Minimale** sur des graphes aléatoires.

Fonctionnalités prévues :
- Génération de 20 graphes aléatoires (n=100, p=0.2)
- Heuristique gloutonne pour une première solution MSC
- Recherche Tabou pour améliorer la solution gloutonne
- Comparaison des résultats (somme des couleurs, temps de calcul, etc.)

Exécution (à terme) :
```bash
python -m src.main
```
